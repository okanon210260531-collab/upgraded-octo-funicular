# RIXON VIDEO WORKS サイト

動画制作・編集サービス向けの静的サイトです。

## 使い方
1. `index.html` をブラウザで開くと確認できます。
2. 会社名・料金・文章は `index.html` を編集してください。
3. 色やデザインは `styles.css` で変更できます。
4. 問い合わせフォームは現在デモです。実際にメール送信するには、Formspree / Netlify Forms / 自社サーバーなどの設定が必要です。

## 構成
- `index.html`：サイト本体
- `styles.css`：デザイン
- `script.js`：スマホメニュー、フォーム確認メッセージ

## 注意
他社映像・アニメ・音楽を使う場合は、権利者の許可や利用条件の確認が必要です。
